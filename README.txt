Ce dossier comprend les modèles utilisés par le simulateur Gazebo.
worlds/
robomovies.world correspond la table de jeu

models/
Contient le fichier de modèle du robot (rosebot) et les divers accesoires de la table de jeu : ampoule, pieds, claps etc.

plugins/


Lancer gazebo sur robomovies.world pour demarer la simulation:
>> gazebo robomovies.world


** Instaler Gazebo
* gazebo s'intalle sous Linux/Ubuntu (pas de version windows). Il est conseillé d'être sous Ubuntu trusty 14.04
